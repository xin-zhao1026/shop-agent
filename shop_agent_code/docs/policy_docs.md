# 七天无理由退货政策

根据消费者权益保护法，自您收到商品之日起七日内，在商品完好、不影响二次销售的前提下，可享受无理由退货。

## 退货条件：

1. 商品外观无损，包装、配件、赠品齐全。
2. 商品未经使用或仅进行不影响二次销售的合理试用。
3. 特定商品（如定制商品、鲜活易腐商品、报纸期刊、音像制品、计算机软件、在线下载或者消费者拆封的音像制品、计算机软件等）不适用七天无理由退货。

## 退货流程：

1. 在订单详情页点击“申请退货”。
2. 填写退货原因并上传相关凭证。
3. 等待商家审核。
4. 商家同意后，按指示寄回商品。
5. 商家收到并确认商品后，将在7个工作日内完成退款。

# 满减优惠券使用规则

## 使用条件：

1. 订单实付金额需达到优惠券指定门槛。
2. 仅限指定商品或品类使用，请查看优惠券详情页说明。
3. 每笔订单限使用一张优惠券，不与其他优惠券叠加。

## 有效期：

请在优惠券有效期内使用，过期作废。

## 注意事项：

1. 退款时，优惠券金额不予退回。
2. 优惠券不可兑换现金。
